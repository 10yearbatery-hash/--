## 2. 디자인 시스템

### 컬러 팔레트

| 이름 | Hex | 용도 |
|------|-----|------|
| Primary Pink | `#FF6B9D` | CTA 버튼, 강조, AI 말풍선 |
| Light Pink | `#FFD6E7` | 배경 카드, 입력 필드 배경 |
| Salmon | `#E8876A` | 히어로 섹션 배경 |
| Warm White | `#FFF5F8` | 페이지 배경 |
| Text Dark | `#1A1A1A` | 본문 텍스트 |
| Text Gray | `#666666` | 서브 텍스트 |
| Border | `#F0D0DC` | 구분선, 카드 테두리 |

### 타이포그래피

- **기본 폰트:** Pretendard (한글 친화적, 웹폰트)
- **폴백:** -apple-system, sans-serif
- **사이즈 스케일:**
  - `text-xs`: 12px (캡션, 힌트)
  - `text-sm`: 14px (서브텍스트)
  - `text-base`: 16px (본문)
  - `text-lg`: 18px (소제목)
  - `text-xl`: 20px (제목)
  - `text-2xl`: 24px (대제목)

### 간격 & 레이아웃

- **최대 너비:** 390px (모바일 기준, 중앙 정렬)
- **페이지 패딩:** px-5 (좌우 20px)
- **카드 패딩:** p-4 ~ p-6
- **버튼 높이:** 52px (CTA 버튼)
- **보더 라디우스:** rounded-2xl (16px, 카드), rounded-full (버튼)

### 그림자

```css
/* 카드 그림자 */
box-shadow: 0 2px 12px rgba(255, 107, 157, 0.1);

/* CTA 버튼 그림자 */
box-shadow: 0 4px 16px rgba(255, 107, 157, 0.3);
```

## 3. 공통 컴포넌트

### 헤더

```
[← 뒤로가기]    [타이틀]    [로그인 버튼]
```
- 홈: 햄버거(☰) + 로그인 버튼
- 채팅/결과: 뒤로가기(<) + 타이틀 + 방 코드

### CTA 버튼 (Primary)

- 배경: Primary Pink (`#FF6B9D`)
- 텍스트: 흰색, font-semibold
- 높이: 52px, rounded-full
- 너비: 전체 너비 (w-full)
- 상태: hover(약간 어둡게), disabled(opacity-50)

### 카드

- 배경: 흰색 또는 Light Pink
- 보더: 1px solid Border
- 라디우스: rounded-2xl
- 패딩: p-5

### 채팅 말풍선

| 유형 | 배경 | 정렬 |
|------|------|------|
| AI (본심 상담사) | Light Pink | 좌측 |
| 유저 | 흰색 + 테두리 | 우측 |

- 타이핑 인디케이터: `...` 애니메이션 (3개 점이 순서대로 나타남)

### 프로그레스 바 (결과 페이지)

- 4단계 원형 스텝 인디케이터
- 완료 단계: Primary Pink 채움 + 체크 아이콘
- 현재 단계: 테두리만 (활성)
- 미완료: 회색

## 5. 모바일 퍼스트 반응형 기준

| 브레이크포인트 | 너비 | 레이아웃 |
|--------------|------|----------|
| 기본 (모바일) | ~390px | 단일 컬럼, 전체 너비 |
| sm | 390px~ | max-w-[390px] 중앙 정렬 |
| md+ | 768px~ | 동일 (모바일 앱 느낌 유지) |

- 모든 터치 타겟 최소 44x44px
- 하단 CTA 버튼은 항상 화면 하단에 고정 또는 충분한 하단 여백
- iOS Safari 하단 안전 영역 고려 (safe-area-inset-bottom)
